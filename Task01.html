Website

A collection of many web pages and media over the internet is called a Website


WWW

World Wide Web : all the public Website or pages that user can access on thair local computers and other devices through the internet


HTTP/HTTPS

Hypertext transfer protocol secure (HTTPS) is the secure version of HTTP, which is the primary protocol used to send data between a web browser and a website. HTTPS is encrypted in order to increase security of data transfer.

Client server

Client-server denotes a relationship between cooperating programs in an application, composed of clients initiating requests for services and servers providing that function or service

Domain

domain name is the address of a website. It's the name you type into a web browser to access that specific website.

Host

any computer connected to a network. It can provide information, applications or services to other hosts or nodes on the network.

Hosting

It is an online service that enables you to publish your website or web application on the internet.

FTP

File Transfer Protocol. FTP is a standard protocol for the transfer of files between computer systems from one client to one server

path

passages or routes
